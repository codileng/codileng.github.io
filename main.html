    Temario

  • HTML5

    • La base de toda página web es HTML

    Aprenderás cómo estructurar una página web y sus elementos

  • CSS

    • CSS nos ayuda a darle estilo y una vista hermosa a nuestra página

    Aquí aprenderemos a darle color, forma, posición y hasta animación a
    los elementos de nuestra página web.

  • Javascript

    • Javascript es el lenguaje de programación que mueve a la web, en este módulo
    veremos las bases de programación y a darle funcionalidad avanzada a nuestra página web.

  • JQuery

    • JQuery nos facilita la creación de funciones con Javascript y nos ayuda a
    programar más, escribiendo menos, lo cual reduce los tiempos de creación de nuestra página

  • Bootstrap

    • Bootstrap facilita de sobremanera el uso de CSS, lo cual agiliza el proceso para llever
    nuestra página web a otro nivel y crear vistas llamativas y profesionales

  • MySQL

    • Para guardar y editar datos necesitamos de una base de datos. Con MySQL podremos darle esa
    funcinalidad a nuestra aplicación web. Por ejemplo, un control de inventarios requiere registrar
    entradas y salidas, así como tener actualidas las existencias de producto.

  • Electron

    • Veremos cómo sacarle provecho a las habilidades aprendidas para crear aplicaciones de escritorio.
    Ejemplos de aplicaciones hechas con Electron son Whatsapp para computadora y VS Code, entre muchas otras.

  • Práctica final

    • Para concluir el curso haremos una página o aplicación para un negocio local, pudiendo cobrar
    por tu servicio y así tener tu primer sueldo como Desarrollador Web, al mismo tiempo ayudaremos al municipio
    acercando la tecnología a pequeños negocios.
